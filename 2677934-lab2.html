Latest Project: To - Do - List App

I created a simple To-Do List App where users can add, edit, and delete tasks. The app allows users to mark tasks as completed and stores the data in the browser's local storage, so the list persists even after refreshing the page. I built it using HTML, CSS, and JavaScript to improve my skills in front-end development and interactive web features. This project helped me learn how to manage user input and build a functional, user-friendly web application. This project is very straightforward and focuses on basic web development, making it perfect for showcasing your ability to build simple, interactive apps.
